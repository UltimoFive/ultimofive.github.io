Game Development Projects

Project Arena

Introduction to Unreal Engine 5 through Course GAM-207 that covers game platforms, interface design, AI, business economics of the game industry, and game marketing. Throughout the course is a project that culminates the conception, design, and prototype of an original digital game.

Project Cyber

A Fast-Paced FPS based on a cybernetic virus outbreak with elements inspired by Doom, Halo, and Bit Trip Runner. As of currently there is a story Synapse that I am writing alongside the development being in the Pre-Alpha stage. Current completion of Pre-Alpha is set for mid-December.

Project Dream Hero

Focus on fundamental idea creation with virtual environments on digital games that covers system theory, cybernetics, interface design, interactive design, and game theory. As well it emphasized how virtual environments function to create experiences, play mechanics, game balancing, and game interaction.

Project Virus Hunter

Provided a technically well-founded introduction to game development within a team focused course. Upon completion, learned the fundamental understanding of the use of sprites animation, and audio in a integrated game environment.
